The Mission

Combating digital misinformation and authenticating media through state-of-the-art Artificial Intelligence.

⚙️ How It Works

1

Frame Extraction

The system robustly parses video uploads (either locally via HTML5 or through backend OpenCV fallback) to sample uniform frames.

2

Facial Recognition

Utilizes MTCNN (Multi-task Cascaded Convolutional Networks) to isolate and crop human faces from the raw digital frames.

3

Deep Learning

Leverages the InceptionResNetV2 neural network architecture to meticulously extract high-dimensional deepfake feature vectors.

4

Final Verdict

A Support Vector Machine (SVM) classifies the features and mathematically computes the confidence percentage for authenticity.

💻 Technology Stack

Frontend

Vanilla JavaScript, Modern Glassmorphism CSS

Backend Engine

Python 3, FastAPI, Uvicorn Server

Machine Learning

TensorFlow, Keras, MTCNN, Scikit-Learn

⚠️ Current Limitations

Face-Dependent Detection

The current pipeline strictly requires facial data to function. If no face is detected in the video frame, the AI cannot perform analysis on background manipulation, synthetic environments, or pure audio deepfakes.

🚀 Future Roadmap

Multi-Modal Analysis (Voice)

Expand detection horizons beyond visual artifacts to spot audio deepfakes and AI voice cloning algorithms.

Photo Authenticity

Introduce single-image processing pipelines to detect GAN-generated static photos and face manipulations.

Enhanced AI Accuracy

Transition to advanced Vision Transformer (ViT) architectures to capture more granular spatial inconsistencies.

Explainable AI (Heatmaps)

Highlight exactly *where* on the face the AI detected synthetic artifacts via visual confidence heatmaps.
