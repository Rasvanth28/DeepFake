📊 Evaluation Breakdown

InceptionResNetV2 Transfer Learning

The model leverages a massive pre-trained dataset (ImageNet), fine-tuned specifically on facial manipulation artifacts. It achieves rapid convergence by freezing the base layers and solely training the dense decision heads.

  • Optimizer: Adam (LR=0.0001)
  • Loss Function: Binary Crossentropy
  • Feature Vector: 128 Dimensions

Support Vector Machine (SVM)

Instead of a raw Softmax layer, the 128-dimensional dense vectors are passed into a highly calibrated Support Vector Machine. This creates a more distinct mathematical boundary separating Real vs. Fake features.

  • Kernel: RBF (Radial Basis Function)
  • C Parameter: 1.0 (Regularization)
  • Probability Maps: Calibrated using Platt Scaling
